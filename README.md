
## Minhas Series
application to manage series watched on shared streamers


## Technology 

Here are the technologies used in this project.


* React JS
* Bootstrap
* Axios


## Services Used

* Github
* Umbler




## Getting started

* Dependency
  - Babel 
  - Eslint
 
  
  
* To install the dependencies.
  - npm install
  
* To run the project.
  - npm start
